Excel to PDF Converter

Simple and efficient Excel to PDF conversion tool for Odoo 16.

Features:

  • Convert single Excel files to PDF
  • Batch conversion of multiple files
  • Cross-platform support (Windows with Excel COM, Linux with LibreOffice)
  • Automatic LibreOffice installation on Linux
  • Error handling and status tracking
  • Simple and user-friendly interface

Usage:

  1. Go to Excel Converter menu
  2. Choose "Excel Files" for single file conversion or "Batch Converter" for multiple files
  3. Upload your Excel files
  4. Click "Convert to PDF" or "Convert All"
  5. Download the generated PDF files
